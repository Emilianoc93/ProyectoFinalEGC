Encriptador

Encriptar textos es similar a incluir un texto en una caja fuerte. Para incluir el texto en la caja fuerte, se usa una clave secreta para encerrar (cifrar) el texto en la caja fuerte. Si desea cifrar un texto, primero ingrese una clave secreta. La clave puede ser cualquier cadena. Luego ingrese el texto a ser cifrado en el campo de entrada en el área de entrada. Si no ingresa una clave secreta, Secure Text usa una clave secreta interna. El texto cifrado se muestra ahora en el área de salida.
